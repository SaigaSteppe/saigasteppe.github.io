Time Needle


A timeline creator that makes a timeline from the data in a csv file.

Download

Requirements

1. To run the program you need to have Java installed.

2. To use this program to create a timeline, enter the information in a csv file.
In the csv file enter the dates and the events, such that each entry has a date (unit: years), seperated by a comma (','), then the event.

Example:

How to use:

1. Open 'Time Needle.jar'.

2. Click on the 'Load CSV...' button and choose your csv file.

3. Click on the 'Generate Timeline' (name the timeline if you wish).
