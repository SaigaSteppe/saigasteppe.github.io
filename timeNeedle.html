Time Needle


Current version: v 1.1.

A timeline creator that makes timelines from the data in csv files.



To view older versions, click here


Patch Notes (v 1.1)

    - New layout
    - Timeline labels rotated vertically to reduce overlapping labels
    - Enabled reading UTF-8 encoded csv files
    - Copy button added
    - Width Slider added

Requirements

1. To run the program you need to have Java installed.

2. To use this program to create a timeline, enter the information in a csv file.
In the csv file; enter the dates and events, such that each entry has a date (unit: years), separated by a comma (','), then the event.

Example:

How to use:

1. Open 'Time Needle.jar'.

2. Click on the 'Load CSV...' button and choose your csv file.

3. Click on the 'Generate Timeline' (name the timeline if you wish).
